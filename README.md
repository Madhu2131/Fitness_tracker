# 🏋️ Fitness Tracker

A Java-based Fitness Tracker web application that helps users monitor workouts, calculate BMI, and manage fitness data efficiently.

## 🚀 Features

* User Registration & Login
* Secure Authentication
* Workout Tracking
* BMI Calculator
* User Dashboard
* Profile Management
* MySQL Database Integration

## 🛠️ Tech Stack

* Java
* JDBC
* Servlets & JSP
* HTML, CSS, JavaScript
* MySQL
* Apache Tomcat
* Eclipse IDE

## ⚙️ Setup Instructions

1. Clone the repository:
   git clone https://github.com/your-username/fitness_tracker.git

2. Import project into Eclipse.

3. Configure Apache Tomcat server.

4. Create a MySQL database.

5. Import the SQL file from:
   database/fitness_tracker.sql

6. Update database credentials in your JDBC code.

7. Run the project on the server.

## 🗄️ Database

* MySQL is used
* Import the provided SQL file before running
